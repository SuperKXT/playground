# countBlinks

[issue #466 of rendezvous with cassidoo.](https://buttondown.com/cassidoo/archive/u1f94f-you-cant-achieve-anything-in-life-without/)

Given a string of `.` and `_` where `_` represents a blink, return the longest gap between blinks.

Examples:

```ts
countBlinks("_..__...._.");
> 4

countBlinks("...._");
> 0
```
